Urbanwood Hotels Quiz - Vercel Deploy Version

This package is prepared for direct deployment to Vercel as a static SPA.

How to deploy:
1. Upload this folder/repo to Vercel
2. Framework preset: Vite
3. Build command: npm run build
4. Output directory: dist/public
5. After deployment, add your domain urbanwoodhotels.com in Vercel Domains

Admin URL:
- /admin

Admin password for this deploy version:
- urbanwood2026

Important:
- This deploy version stores admin edits and form submissions in the browser localStorage.
- That means submissions/config are device-browser specific, not shared across all visitors.
- This is good for preview/demo deployment.
- If you want real shared backend data for production, the next step is connecting a database.
